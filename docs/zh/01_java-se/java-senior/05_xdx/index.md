## 1.2 查找算法

### 1.2.1 概述

* `查找算法`有`基本查找`、`二分查找`、`插值查找`、`斐波那契查找`、`分块查找`、`哈希查找`以及`树表查找`等。
* 本次，将重点讲解`基本查找`、`二分查找`和`分块查找`。

### 1.2.2 基本查找（顺序查找）

* 顺序查找（线形查找），属于无序查找算法。从数据结构线的一端开始，顺序扫描，依次将遍历到的结点与要查找的值相比较，若相等则表示查找成功；若遍历结束仍没有找到相同的，表示查找失败。
